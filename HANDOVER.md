## Adding New Photos

### Method 1: Manual
1. Upload image to S3:
   ```bash
   aws s3 cp /path/to/photo.jpg s3://gp-derelict/filename.jpg
   ```
2. Add entry to `_data/gallery.yml`:
   ```yaml
   - filename: filename.jpg
     caption: YOUR CAPTION IN UPPERCASE
     alt: Description for accessibility
     category: CategoryName
   ```
3. Commit and push:
   ```bash
   git add _data/gallery.yml
   git commit -m "Add new photo: caption"
   git push
   ```

### Method 2: Using gallery.json (Client-Side Fetch)
For automatic display without GitHub rebuilds:
1. Create `gallery.json` in S3 bucket
2. Add JavaScript to `index.md` to fetch and render
3. Update JSON when adding new images

## Jekyll Commands

Build locally (requires Ruby & Jekyll):
```bash
bundle install
jekyll serve
```

Or with Docker:
```bash
docker run --rm -v $(pwd):/srv/jekyll -p 4000:4000 jekyll/jekyll jekyll serve
```

## S3 Configuration

Bucket `gp-derelict` is configured for:
- Static website hosting (index.html)
- Public read access for images

To add images:
```bash
aws s3 cp local-file.jpg s3://gp-derelict/remote-file.jpg
```

To list all images:
```bash
aws s3 ls s3://gp-derelict/ --recursive
```
